# Dilla Extism with NodeJS

Minimal usage of Dilla WASM with NodeJS leveraging [Extism](https://github.com/extism/js-sdk), visit us at [https://dilla.io](https://dilla.io)

## Build

```shell
make build
```

## Run Dilla render

```shell
make render
```

## Run Dilla describer

```shell
make describe
```
