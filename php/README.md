# Dilla Extism with PHP

Minimal usage of Dilla WASM with PHP leveraging [Extism](https://github.com/extism/php-sdk), visit us at [https://dilla.io](https://dilla.io)

## Build

```shell
make build
```

## Run Dilla render

```shell
make render
```

## Run Dilla describer

```shell
make describe
```
