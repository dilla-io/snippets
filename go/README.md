# Dilla Extism with Go

Minimal usage of Dilla WASM with Go leveraging [Extism](https://github.com/extism/dotnet-sdk), visit us at [https://dilla.io](https://dilla.io)

## Build

```shell
make build
```

## Run Dilla render

```shell
make render
```

## Run Dilla describer

```shell
make describe
```
